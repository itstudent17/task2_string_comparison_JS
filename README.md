# task2_string_comparison_JS

Есть три вида скобок: {}, () и [].

Дана входная строка, состоящая из этих символов, надо определить, корректна ли строка, т.е. для каждой закрывающей скобки должна быть своя открывающая.

isCorrect('') === true;
isCorrect('()') === true;
isCorrect('{(})[]') === false;
